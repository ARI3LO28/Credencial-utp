# Scrum - Credencial Digital UTP

## Roles
- Product Owner: Docente / responsable academico.
- Scrum Master: Lider del equipo.
- Development Team: Colaboradores del repositorio.

## Product Backlog
- Autenticacion con correo institucional y contrasena.
- Credencial digital con matricula, nombre, carrera, estado y fotografia.
- QR de validacion generado por servicio externo.
- Consulta de notas academicas.
- Consulta de prestamos de biblioteca.
- Actualizacion de fotografia en Firebase Storage.
- Consulta de beneficios estudiantiles.
- API REST propia con Express y Firestore.

## Sprint sugerido
- Sprint 1: Arquitectura MVC, Firebase Authentication y rutas privadas.
- Sprint 2: Credencial, QR, consultas academicas y biblioteca.
- Sprint 3: Fotografia, beneficios, API Express y evidencias.

## Definition of Done
- Codigo en rama feature.
- Validacion local con `npm run lint`.
- Pull Request hacia `develop`.
- Revision por colaborador.
- Merge a `develop` y posteriormente a `main`.
